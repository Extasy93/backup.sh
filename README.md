Setup
-----
1. Connectez-vous en root sur votre serveur via SSH 
2. Installez backup.sh
		wget https://raw.githubusercontent.com/Extasy93/backup.sh
    
3. chmod +x backup.sh
4. éxécuté le script avec la commande ./backup.sh

Et vous êtes prêt.
